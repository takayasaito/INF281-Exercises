  \part Calculate the following basic evaluation measures for the oval representation. Round off the answer to two decimal places if necessary.
  
\bigskip 
Accuracy
$= \dfrac{TP+TN}{P+N} $

\bigskip 
Error rate
$= \dfrac{FP+FN}{P+N} $

\bigskip 
Sensitivity
$= \dfrac{TP}{P} $

\bigskip 
Specificity
$= \dfrac{TN}{N} $
